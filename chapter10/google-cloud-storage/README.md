# Storage

## 버킷 생성
* **버킷 이름 지정** : 하고 싶은 이름으로 해주세요.
* **데이터 저장 위치 선택 - 위치 유형** : Region
* **데이터 저장 위치 선택 - 위치** : asia-northeast3
* **데이터의 기본 스터리지 클래스 선택** : Standard
* **객체 액세스를 제어하는 방식 선택** : 세분화된 액세스 제어
